# 엔지니어링 지식 DB

물류설비(STK·AGV/AMR·CNV·LIFT·PORT/OHT·ROBOT) 엔지니어링 데이터를 표준·논문·기술문헌에서
조사하여 플랫폼 내 지식 DB(`/api/v1/knowledge`)로 적재했다 (2026-06 조사, 출처 링크 포함 17건).

## 와이어로프 수명예측 PRO 툴 산식

```
줄당 장력 S = (가반하중 + 운반구 자중) × g × 동적계수 ÷ 줄수 ÷ η^시브수 + 로프자중
최소파단력 MBF = K × d² × R0 / 1000  [kN]   (EN 12385-4, 6×36: K=0.356)
안전율 SF = MBF / S   (법규 요구: 화물 직접지지 5 이상)

폐기기준 도달 굽힘수 N = 300,000 × (D/d ÷ 25)^2.5 × (SF ÷ 5)² × k_env × k_rope
  k_env: 청정 1.0 / 일반 0.85 / 분진 0.65 / 부식성 0.45
  k_rope: 일반 1.0 / 회전저항 0.6
수명(년) = N ÷ (시브수×2+1) ÷ (일일 사이클 × 연간 가동일)
계획교체 권고 = 수명 × 0.8
```

Feyrer 시험 데이터의 경향(장력·D/d 민감도)을 반영한 보수적 간이식이며, 실측 단선 추세가
축적되면 N0·지수를 보정한다(Phase 4 RUL 고도화).
