# 정기 촬영 기반 설비 상태 감시 시스템

사진/동영상으로 설비를 정기 촬영하여 **볼트 풀림 · 와이어 소선 불량 · 설비 파손 · 레일 단차**
등의 이상 상태를 자동 감지하고, 이상 발생 시 이슈를 자동 생성해 알려주는 시스템.

## 1. 운영 개념

```
촬영 포인트 등록 ─► 기준 이미지(골든샘플) 등록 ─► 정기 촬영 업로드(사진/동영상)
                                                      │
              위치 정합 → 변화 감지 → 타입별 전용 검출 → 이상점수·판정
                                                      │
        OK ──────────── 추세 기록                      │
        CHECK/NG ─► 이슈 자동 생성(대시보드 '오늘의 조치 대상') + 생애주기 기록
                    + 변화영역 박스 오버레이 이미지 제공
```

- **촬영 포인트**: 설비의 감시 지점. 동일 지그·구도·조명으로 촬영하는 것이 표준
  (`location_note` 에 촬영 표준 기록). 촬영 주기를 설정하면 지연 시 "촬영 지연" 표시.
- **기준 이미지**: 정상 상태에서 촬영한 골든샘플. 부품 교체·정비 후에는 기준 재등록.
- **동영상**: 균등 간격 프레임 추출 후 각각 분석, 최악(점수 최대) 프레임으로 판정.
  압축 열화로 정밀도가 낮으므로 정밀 측정은 정지 사진 권장.

## 2. 분석 파이프라인

| 단계 | 방법 | 목적 |
|------|------|------|
| 위치 정합 | 위상상관(FFT phase correlation), ±64px | 카메라 미세 흔들림·재현 오차 보정 |
| 변화 감지 | 박스블러 후 국부 차이맵 → 16px 블록 단위 변화 플래그 → BFS 클러스터링 | 변화율(%)·변화 블롭 수·최대 블롭 크기 |
| 이상점수 | 변화율 기반(0~60) + 타입별 검출 가산(최대 80) | 0~100 표준화 점수 |
| 판정 | warn_score(기본 30) / ng_score(기본 60) + **추세 룰**(이력 평균+3σ 초과 시 CHECK) | OK / CHECK / NG |

## 3. 타입별 전용 검출기

| 유형 | 대상 | 검출 방법 | 판정 파라미터 |
|------|------|----------|--------------|
| `BOLT` | 볼트 풀림 | **합마크(매칭 라인) 어긋남**: 변화 영역에 한정한 방향 히스토그램으로 기준 마크각 vs 현재 마크각의 차이 추정. 고정 에지(육각 머리·배경)는 변화 영역 밖이라 배제 | `rotation_limit_deg` (기본 8°) — 각도는 근사치이며 안전측 과대평가 경향. 합마크 점검의 본질은 "어긋남 여부" |
| `WIRE` | 소선 단선 | 로프 윤곽의 **신규 돌출**: 에지 밀도 증가량 + 소형 변화 블롭(≤4블록) 개수 | 돌출 3개소↑ 또는 에지 +1.2%p → NG |
| `SURFACE` | 파손·크랙 | 기준에 없던 **대형 변화 블롭**(≥6블록) 검출 | 1개소 이상 → NG |
| `RAIL` | 이음부 단차 | 각 열의 최강 수평에지 y위치 → **조인트 좌/우 중앙값 차** → mm 환산 | `mm_per_px`, `step_limit_mm` (기본 0.5mm — OHT 레일 L&L 관리기준 0.3mm 권장) |
| `GENERIC` | 범용 | 변화율·블롭 기반 | `sensitivity` (기본 3.0) |

전제 조건 (현장 표준화가 검출 성능을 좌우):
- **볼트**: 체결 후 볼트머리→너트/플랜지를 잇는 합마크 도색 (산업 표준 관행)
- **레일**: 조인트 정면, 레일 헤드 상면 에지가 수평으로 보이는 구도 + mm/px 캘리브레이션
- 공통: 촬영 지그 고정, 조명 일정 (정합은 평행이동만 보정 — 회전·배율 변화는 재촬영)

## 4. 알림·연계

- NG → **이슈 자동 생성 (severity HIGH)**, CHECK → MID. 대시보드 '오늘의 조치 대상'에 즉시 표출
- 설비 생애주기 이력에 자동 기록 (NG=BM 단계, CHECK=PM 단계)
- 변화 영역 박스 오버레이 이미지 (`/uploads/...`) — 현장 확인 시 어디를 볼지 바로 안내
- 촬영 지연 포인트 목록 API (`GET /vision-monitor/points/due`) — 정기 촬영 누락 감시

## 5. 데모 및 검증

`python -m app.seed` 또는 UI [데모 포인트 생성] 버튼 → 합성 이미지 4종 세트 생성:
볼트 합마크 16° 회전 / 와이어 돌출 소선 3개소 / 레일 단차 0.8mm / 플레이트 크랙 — 모두 NG 검출 확인.

## 6. 고도화 로드맵

1. (현재) 골든샘플 비교 + 룰 기반 — 설명 가능, 즉시 적용
2. 고정 카메라 + 자동 주기 촬영(RTSP 연동) — 무인화
3. NG 이미지 축적 → 딥러닝 결함 분류(전이학습), 세그멘테이션 정밀화
4. FDC 센서(진동·전류)와 융합 판정, 추세 기반 잔존수명(RUL) 추정
