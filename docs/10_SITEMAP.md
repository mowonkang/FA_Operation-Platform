# 시스템 사이트맵

플랫폼 내 [시스템 → 사이트맵] 화면과 동일한 구성. (UI 에서 각 화면으로 바로 이동 가능)

```
FA Operation Platform
│
├─ 모니터링 (사이트 범위 — 상단바 사이트 셀렉터 적용)
│   ├─ 대시보드          KPI 4종 · 오늘의 조치 대상 · 주간 BM/PM/알람 분포
│   ├─ FDC 트렌드/알람    센서 트렌드 · 레벨/스파이크/드리프트 · 알람→BM
│   └─ 이슈 관리          설비(기구/전장/제어/인터락)·시스템(CIM/MCS/RTD)·안전
│
├─ 라이프사이클 (사이트 범위)
│   ├─ 라이프사이클 맵     투자→제작→셋업→양산→폐기 (편집 가능, 모듈 연결)
│   ├─ 프로젝트           투자/구축 프로젝트 CRUD (삭제 = 관리자), 예산·견적 요약
│   ├─ 투자·견적 분석      견적 자동분류/분석 · ERRC · 업체 비교
│   ├─ 설비 마스터/이력    생애주기 타임라인 · Install Parameter 버전
│   └─ 워크플로우          7종 표준 절차 · DR 데이터팩(운영이력→설계 환류)
│
├─ 정비 운영 (사이트 범위)
│   ├─ PM 표준/오더        표준 점검항목(전사) · 주기 오더 · 자동판정
│   ├─ BM 고장정비         보고→원인→조치 · 파츠 차감 · L&L 연계
│   ├─ 비전 측정           정기 상태감시(볼트/와이어/파손/레일) · 순회 동영상 · 수동 측정
│   └─ 스페어 파츠         재고 · BOM · MTBF 권장재고
│
├─ 전사 공통 표준 (사이트 무관)
│   ├─ 계산 툴 (PRO)       7종 수명/용량 계산기 + 산정 기준·표준 근거
│   ├─ 지식 DB·표준        ISO/FEM/DIN/법규/논문 지식 (출처 포함)
│   └─ Lesson & Learn      전사 풀 → 전 사이트 전파 → PM 표준 반영
│
├─ 시스템
│   ├─ 데이터 관리 (엑셀)   9개 항목 양식/가져오기/내보내기
│   └─ 사이트맵            전체 구성·데이터 흐름
│
└─ 설정 (⚙ 상단바)         테마 4종(라이트/다크/페트롤/고대비) · 표시 밀도 · 역할(관리자)
```

## 엑셀 입출력 지원 항목 (9종)

| 항목 | 키 | 방식 | upsert 기준 |
|---|---|---|---|
| 설비 마스터 | equipments | 생성/갱신 | asset_code |
| 스페어 파츠 | parts | 생성/갱신 | part_no |
| 모델별 BOM | bom | 생성/갱신 | model_code × part_no |
| PM 표준 점검항목 | pm_standards | 생성/갱신 | model_code × item_no |
| BM 고장 이력 | bm_reports | 추가 전용 | — |
| 이슈 | issues | 추가 전용 | — (SAFETY 자동 HIGH) |
| FDC 센서 | fdc_sensors | 생성/갱신 | asset_code × name |
| Lesson & Learn | lessons | 추가 전용 | — (전 사이트 자동 전파) |
| 프로젝트 | projects | 생성/갱신 | code |

API: `GET /excel/template/{key}` · `GET /excel/export/{key}` · `POST /excel/import/{key}`
