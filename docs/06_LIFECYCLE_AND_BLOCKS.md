# 전체 라이프사이클 체계와 블럭 아키텍처

## 1. 라이프사이클 (시스템에서 편집 가능)

기본 구성 (DB 마스터 — 추가/이름변경/수정/삭제/순서변경 전부 가능):

```
1. 투자      기본품의 → 사양서 작성 → 견적 입수 → ERRC 검토 → 구매 Nego → 업체 선정
2. 제작      DR → 가공 → 조립 → 전장 → 제어 → FAT → FOB
3. 셋업      반입/설치 → Install Parameter → 알람 관리 → 설비 이슈(기구/전장/제어/인터락)
             → 시스템 이슈(CIM/MCS/RTD) → 안전 점검 → 안정화 판정
4. 양산      PM → BM → 자동 스캐닝(비전) → FDC → 스페어파츠 → L&L
5. 폐기/이설 오버홀/수명연장 검토 → 이설 → 매각/폐기   ← 플랫폼 제안 단계
```

- 마스터 API: `GET /lifecycle-config/map`, `POST/PATCH/DELETE /lifecycle-config/phases|processes`
- UI: [라이프사이클 맵] 화면의 편집 모드 — 카드(블럭) 단위로 이름/설명/순서/삭제, 단계·프로세스 추가
- 각 프로세스는 `module_key` 로 플랫폼 모듈(블럭)에 연결 — 카드에서 해당 화면으로 바로 이동

## 2. 블럭 아키텍처 (계속 확장 가능한 구조)

```
프로세스(라이프사이클 카드) ──module_key──► 기능 블럭
기능 블럭 = 백엔드 라우터 1개 + 프론트 페이지(or 패널) 1개
```

### 새 블럭 추가 절차 (4단계)
1. **백엔드**: `app/routers/<신규>.py` 작성 → `main.py` 에 `include_router` 1줄
2. **프론트**: `src/pages/<신규>.tsx` 작성 → `App.tsx` 메뉴/라우트 2줄
3. **연결**: 라이프사이클 맵 편집모드에서 프로세스의 `module_key` 지정 (또는 PATCH API)
4. (선택) 워크플로우 템플릿 `workflow_templates.py` / 지식 DB에 관련 항목 추가

### 엔지니어링 계산기 블럭 (코드 없는 수준의 확장)
`Engineering.tsx` 의 `PRO_BLOCKS` 배열에 설정(입력 필드·엔드포인트·곡선 정의)만 추가하면
새 계산기 블럭이 생성된다. 백엔드는 `services/engineering.py` 에 함수 1개 + 라우터 3줄.

현재 블럭: 와이어로프 PRO(체결방식·민감도 곡선 3종) / 모터 용량(주행·권상) /
컨베이어 출력 / 리프체인 수명(신율 추세) / 베어링 L10 / 배터리 / 휠 / (구)와이어로프

## 3. 견적서 자동 데이터 분석 (투자 블럭)

업로드: CSV/XLSX — 헤더 행 자동 탐색(품명/수량/단가/금액 키워드), CP949/UTF-8 자동 인식,
합계/소계/부가세 행 자동 제외.

자동 분석 항목:
| 분석 | 방법 | 활용 |
|------|------|------|
| 항목 자동 분류 | 키워드 → 기구/구동/전장/제어/SW/설치/기타 | 원가구조 비교 기준 통일 |
| 원가 구조 | 카테고리별 금액·비중, 설치비>20%·SW>15% 경고 | 사양 적정성 검토 |
| 계산 오류 | 수량×단가 ≠ 금액 검출 | Nego 전 정정 요청 |
| 이상 단가 | 카테고리 내 z-score>2 | 단가 근거 요청 |
| 중복 의심 | 정규화 명칭 2회 이상 출현 | Eliminate 후보 |
| 파레토 | 누적 80% 도달 상위 항목 수 | 협상 집중 대상 선정 |
| ERRC 태깅 | 항목별 E/R/R/C 지정 → 절감액 추정(E 전액+Reduce 30%) | 사양 최적화 기록 |
| 업체 비교 | 총액·카테고리 매트릭스·공통 항목 단가차(spread)·Nego 여지 금액 | 데이터 기반 구매 협상 |

## 4. 이슈 관리 블럭

도메인: 설비(기구 MECH/전장 ELEC/제어 CONTROL/인터락 INTERLOCK),
시스템(CIM/MCS/RTD), 안전(SAFETY — 자동 HIGH), 기타.
이슈 종결 시 설비 생애주기 이력에 자동 기록. 미결 HIGH 이슈는 안정화 판정 보류 기준.
